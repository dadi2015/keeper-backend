## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run after starting

```
1. Log in with administrator rights in the application.
2. Run a query to create default categories
     http://localhost:5000/category/create-default-category
```
